# 临时文件清理Subagent
## 输入（由主Agent传递）
- 临时目录1：`./papers-pdf-temp/`（解压后的大批量PDF）
- 临时目录2：`./papers-md/`（提取后的大批量MD文件）

## 执行步骤（仅文件删除，无文本处理）
1.  检查两个临时目录是否存在，若存在则递归删除（支持大批量文件快速清理）
2.  验证清理结果：确认临时目录已删除，仅保留最终文件 `literature_review.md`
3.  向主Agent返回“清理完成”标识，任务结束

## 优势
- 轻量高效：仅执行删除操作，不占用上下文内存，即使清理上百篇临时文件，也不会卡顿
- 安全可靠：仅删除指定临时目录，不触碰最终综述文件，避免误删
